gfxCardStatus v1.7 Released! Full 9400M/9600M GT Support, Lots More…

May 14th, 2010 | Comments

That’s right! v1.7 has been released! Full support for indicating status of 9400M/9600M GT systems (as well as the switching that worked before) is built in to v1.7.

The rest of the users on i5/i7 systems will see three new things: Intel Only mode, NVIDIA Only mode, and Dynamic Switching mode. The first two are obvious, and the latter hands dynamic switching control back to OS X. I have also completely eliminated the delay associated with opening the menu bar item when the dependent process list was being generated, while the 330M was the active GPU. Cool, huh?!

Go get it! Let me know if you run into any issues. I’m aware of the bug where it occasionally switches back to the 330M when you’re on Intel Only mode and you quit all apps that would normally use the 330M. This will be fixed in a maintenance release in the future.

Have fun!

gfxCardStatus and 2009 MacBook Pros With NVIDIA 9400/9600 GPUs

May 11th, 2010 | Comments

Many users have reported that gfxCardStatus works even on the 2009 MacBook Pros! I don’t have one to test with, but I can only assume that this is indeed the case. It wil continually show the “n,” as the detection for which GPU is in use isn’t built in yet, but switching works, without logging out. I’ll build the detection into a future version, hopefully sooner rather than later! I envision an icon change: “d” for discrete, and “i” for integrated. This will provide greater consistency across all platforms.

gfxCardStatus: MacBook Pro I5/i7 GPU Status

Apr 24th, 2010 | Comments

I just got my hands on a brand new 17” MacBook Pro with the 2.66GHz Intel Core i7 in it, and the NVIDIA GeForce GT 330M graphics card. The thing is screaming fast! I was a bit annoyed by the fact that you couldn’t easily view which GPU was in use…but now, there’s an app for that! Specifically, gfxCardStatus. It’s of my own creation. You can download it here: http://codykrieger.com/gfxCardStatus

gfxCardStatus is a menu bar application that automatically updates when your display configuration changes (e.g. plugging in an external display, etc.). The “i” icon signifies that the Intel HD Graphics are in use, and the “n” signifies that the NVIDIA GeForce GT 330M is in use.

Please note that this app will not work with any generations of the MacBook Pro prior to the most recent model with the i5s and i7s that have the 330M. Any dual-GPU MacBook Pro is supported (late 2008 to current generation).

Until I set up a permanent webpage for gfxCardStatus, discussion of it lives in this MacRumors Forums thread: http://forums.macrumors.com/showthread.php?t=901521&page=5#122 gfxCardStatus’ official homepage: http://codykrieger.com/gfxCardStatus/

Enjoy, and let me know what you think!
