--- layout: post title: gfxCardStatus is officially open-source. published: true ---

You heard right! I have open-sourced the code for gfxCardStatus. It can be found on GitHub at http://github.com/codykrieger/gfxCardStatus. Create a fork, go nuts, and feel free to send the comments and suggestions my way!

v1.7 is almost upon us, which will support toggling automatic switching between GPUs, or manually locking the machine on one or the other GPU. Cheers!
